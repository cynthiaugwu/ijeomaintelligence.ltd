Data & ML Engineer

Contract — 3 months
Remote — Nigeria

About Ijeoma

Ijeoma Intelligence is a real-time safety platform for individuals and organisations operating in high-risk environments. Initially launching in Nigeria, Ijeoma aims to serve the world's most complex and challenging operating environments.

The Role

We need an engineer to own the data infrastructure, backend systems, and machine learning pipeline that power the platform. You will work directly with the founder, who will provide detailed technical specifications under contract.

What You'll Work On

  • Design and maintain the backend data architecture across multiple data sources
  • Build and harden data ingestion pipelines with classification, validation, and deduplication
  • Build and deploy machine learning models from provided specifications
  • Develop the enterprise web platform for organisational clients
  • Build backend services that process and serve data to the mobile application
  • Define and maintain API contracts consumed by the mobile application
  • Ensure structured event capture across all system touchpoints

Full technical specifications, system architecture, and data strategy will be shared under NDA.

Requirements

  • 3+ years backend engineering with production systems
  • Strong database design skills — Firestore, PostgreSQL, or equivalent
  • Experience building data pipelines and ingestion systems
  • Advanced Python proficiency
  • Experience with machine learning model building and deployment
  • React or Next.js for web platform development
  • Comfortable with geospatial data and real-time systems
  • Security-conscious — API hardening, role-based access, data protection
  • Self-directed and able to make sound architectural decisions independently

Nice to Have

  • At least 1 enterprise platform built and deployed
  • Experience with safety, security, intelligence, or logistics platforms
  • Web scraping and data quality systems
  • Building for emerging market infrastructure constraints

Compensation & Working Arrangement

Compensation
Full engagement (3 months) · Based on experience
  • Fully remote — Nigeria based
  • Direct reporting to the founder
  • Weekly syncs, two-week delivery sprints
  • GitHub for code, Slack for communication
  • Full codebase and documentation access from day one (shared under NDA)
  • Possibility of extension based on performance

How to Apply

Send the following to careers@ijeomaintelligence.com

  • A short note (max 300 words) on how you would approach designing a backend data architecture for a real-time platform
  • Your availability and expected compensation
  • Link to at least two GitHub repos of projects you built (not tutorial clones)
